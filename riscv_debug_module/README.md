# RISC-V Data Memory Module

This project implements a **Data Memory module for a RISC-V processor**
using **SystemVerilog**. The design was simulated using **Xilinx Vivado**.

## Description
The module handles memory read and write operations for the RISC-V
processor. It supports 64-bit data storage and instruction-based
memory access.

## Files
- `dm_m.sv` – SystemVerilog implementation of the data memory module
- `dm_tb.sv` – Testbench for simulation and verification

## Tools
- SystemVerilog
- Xilinx Vivado
