# Evaluation of Open-Source Control Libraries (Python, SciPy, Julia)

## Description
This project evaluates the performance and capabilities of open-source control system libraries by implementing classical control system experiments that were originally performed using MATLAB in an Advanced Control Systems laboratory.

The same experiments were reproduced using open-source tools in **Python and Julia**, and all simulations and analyses were carried out using **Jupyter Notebook**.

The main goal of this project was to investigate whether open-source ecosystems can effectively replace MATLAB for control system modeling, simulation, and analysis.

The project includes implementation of several control engineering tasks such as modeling dynamic systems, analyzing step responses, implementing state-space representations, designing controllers, and implementing observers. The results obtained from the open-source implementations were then compared with the MATLAB-based reference results.

This work demonstrates that modern open-source tools can be effectively used for control engineering analysis and research.

---

## Key Features

- Implementation of control system experiments using open-source tools  
- Step response analysis of dynamic systems  
- State-space modeling and simulation  
- Controller design and observer implementation  
- Comparison of results with MATLAB-based experiments
