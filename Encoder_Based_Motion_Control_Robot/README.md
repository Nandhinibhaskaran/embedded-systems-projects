# Encoder-Based Motion Control Robot

This project implements a mobile robot with **closed-loop speed control** using **rotary encoders**, **Raspberry Pi**, and **Python**. Encoder feedback is used to maintain accurate wheel speed and improve motion precision.

## Description

The robot continuously reads encoder pulses to calculate wheel speed. The measured speed is compared with the desired speed, allowing the controller to adjust motor outputs for precise and stable movement.
