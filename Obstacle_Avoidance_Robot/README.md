# Obstacle Avoidance Robot (Raspberry Pi)

This project implements an autonomous obstacle avoidance robot using **Raspberry Pi**, **Python**, and an **HC-SR04 ultrasonic sensor**. The robot continuously detects nearby obstacles and changes its direction automatically to avoid collisions.

## Description

The robot measures the distance to obstacles using an ultrasonic sensor. When an object is detected within a predefined threshold, the Raspberry Pi controls the motors to stop, turn, and continue moving safely.

## Hardware

- Raspberry Pi
- HC-SR04 Ultrasonic Sensor
- DC Motors
- Motor Driver
- Chassis
- Battery Pack

## Software

- Python
- Raspberry Pi GPIO
- PWM Motor Control

## Features

- Real-time obstacle detection
- Autonomous navigation
- Distance measurement using ultrasonic sensor
- Motor speed and direction control
